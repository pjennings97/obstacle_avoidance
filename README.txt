==============================================================
                     obstacle_avoidance
==============================================================
move_baseと接続して、ロボットが障害物にぶつからずに動き続けるよう、
ロボットが到達できるゴールをpublishし続けるノードを含んだパッケージです。

==============================================================
                         ビルドの仕方
==============================================================
catkinのワークスペース（catkin_wsなど）のsrcフォルダーに入れてから、
「catkin_make」でビルドします。

==============================================================
                         　 使い方
==============================================================
「roslaunch obstacle_avoidance obstacle_avoidance.launch」で
起動させたら、gazeboとrvizが開きます。
move_baseによる位置推定の初期化には最初となる推定が必要なので、rvizの
「2D Pose Estimate」でロボットのおおよその位置を推定して設定します。
位置を設定してから「2D Nav Goal」で最初のゴールを設定したら、ロボットが
動き出します。ロボットがゴールに到達したら、このパッケージのノードが
自動的にゴールを設定し続けるので、ロボットが動き続けます。
